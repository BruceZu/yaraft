# yaraft

yaraft is a migration of [etcd/raft](https://github.com/coreos/etcd/tree/master/raft) from golang to C++11.

## Features

- [x] Leader Election
- [x] Log Replication
- [ ] Prevote
- [ ] CheckQuorum
- [ ] Log Compaction / InstallSnapshot
- [ ] Flow Control
- [ ] Restart
- [x] Single-Node Cluster
- [ ] Read-only
- [ ] Leader Transfer
- [ ] Leader Stickiness
